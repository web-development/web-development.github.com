Zentrieren einer Box

Achtung: text-align: center funktioniert nur für Text (no-na!), nicht für einen ganzen Block wie div.

In diesem Beispiel wird ein div-Tag mit Hilfe von margin auto zentriert. Der div-Tag hat eine Breite kleiner 100%. Der verbleibende Platz wird automatisch gleichmäßig verteilt.

margin: 0px auto;
width: 60%;

you can only use text-align: center on text (duh!) not on a block-level element like div. The trick is to use the margin of the block-level element. With the value auto the available space left and right of the element ist automatically distributed equally.
