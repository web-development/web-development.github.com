Achtung: während Indexes von MySQL und MariaDB immer unterstütz werden, sind Constraints und Transaktionen nur in manchen “Database Engines” verfügbar.

Engines

mysql> show engines;

| Engine     | Support  | Comment                                                        |
+------------+----------+----------------------------------------------------------------+
| MyISAM     | DEFAULT  | Default engine as of MySQL 3.23 with great performance         |
| MEMORY     | YES      | Hash based, stored in memory, useful for temporary tables      |

| BerkeleyDB | DISABLED | Supports transactions and page-level locking                   |
| BLACKHOLE  | NO       | /dev/null storage engine (anything you write to it disappears) |
